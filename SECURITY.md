# Security Policy

## Supported Versions

The following versions of Portiq are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v0.3.0  | :white_check_mark: |
| < v0.3.0| :x:                |

## Reporting a Vulnerability

We take the security of Portiq seriously. If you believe you have found a security vulnerability in Portiq, please report it to us by following these steps:

1.  **Do not** open a public issue.
2.  Send an email to [security@portiq.app](mailto:security@portiq.app) (replacement for actual email if available).
3.  Include as much detail as possible, including steps to reproduce the vulnerability and any potential impact.

We will acknowledge receipt of your report within 48 hours and provide a timeline for a fix within 5 business days.

### What to expect
- A response acknowledging your report.
- A private discussion about the vulnerability.
- A public disclosure once the vulnerability is fixed or a workaround is available.
