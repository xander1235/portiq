# Support Guidelines

Thank you for using Portiq! If you need help, please follow these guidelines to get support.

## 🔍 Check Existing Resources
Before reaching out, please check the following resources:
- [README.md](README.md) for general information and setup.
- [GitHub Issues](https://github.com/xander1235/portiq/issues) to see if your problem has already been reported.
- [GitHub Discussions](https://github.com/xander1235/portiq/discussions) (if enabled) for general questions.

## 🐛 Bug Reports
If you've found a bug, please open a [Bug Report](https://github.com/xander1235/portiq/issues/new?template=bug_report.md) with reproduction steps.

## ✨ Feature Requests
Have an idea? Open a [Feature Request](https://github.com/xander1235/portiq/issues/new?template=feature_request.md).

## 💬 Community & Discussions
For general questions, "how-to" advice, or architectural discussions:
- Start a [Discussion](https://github.com/xander1235/portiq/discussions).
- Join our [Discord/Slack/etc.] (Update with link if available).

## 🔐 Security Issues
Please see our [Security Policy](SECURITY.md) for reporting vulnerabilities.
