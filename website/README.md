# Portiq Website

This is a standalone static website for Portiq. It is intentionally separate from the app source so you can host it independently on a free static hosting platform.

## Files

- `index.html` - the landing page
- `styles.css` - all styling for the page
- `assets/portiq-logo.png` - local logo used by the site

## Free Hosting Options

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## Quick Local Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

Example:

```bash
npx serve website
```

## Before Publishing

Update these placeholder links in `index.html`:

- `https://example.com/download`
- `https://github.com/`
